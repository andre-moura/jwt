# JWT



## How to get started

Clone the repository
```bash
git clone https://github.com/andre-moura/jwt.git
```

Create virtual environment
```bash
python -m venv env
```

Enter virtual environment
```bash
.\env\Scripts\activate
```

Install libraries
```bash
pip install -r requirements.txt
```

Start the application
```bash
flask run
```
